Graphic Designing

Learn UI/UX Design through this course!

Behind every successful application or website is a good user interface.
Designing an impressive UI/UX requires substantial graphic designing skills.
In this program, students will use Figma to help improve their UI/UX.


LEARNING OUTCOME

      

Understand the design process: understand user requirements, create sketches, apply a palette and display in wireframe prototypes.

      

Develop UI/UX knowledge and concepts like functionality, visual aesthetic, accessibility, branding and usability.

      

Practice User Interface development through Figma, a tool for prototyping, animating, designing, building and exporting.

      

Replicate preexisting Web Application UI and improvise it with better criteria for greater understanding of UI/UX Development.

Certification

Upon completion of the course, an official certificate will be issued.

Flexible Timings

Create your own schedule, learn at your own pace.

Duration

This course has an estimate duration of 30 days.

Software

Figma, PixlrE
