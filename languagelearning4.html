HTML Programming

Basic Element
Writing Text:
When we want to write text, there a series of different tags that we can use based on our requirements. Just like when we write in our copybooks, we use different sizes, bold text, italic, light, etc. For these different types of text, there are different tags to be used. Heading text: for writing heading texts, like titles and subtitles we use the heading tags. There are 6 different heading tags to choose from, from h1 to h6. By default, h1 is the largest size text and h6 is the smallest. You can write these heading tags like this and that is how it will look when the output is shown.





Paragraph text: Writing basic text anywhere on the HTML body is very simple. It is probably the smallest tag available in HTML. It is the paragraph tag. The paragraph tag has default settings with a font-size of 1em, which is equivalent to 16px. Here is how it looks like.





Extra Lines: Some of us love hitting enter on our keyboard after we write down a point or paragraph. There is a very simple tag in HTML to do the exact same. It is called br, for line break. You can use it in the middle of a lot of tags as well, just to keep your work neat, nice, and organised.

Separating Sections: There are 2 main tags that you should know if you ever want to divide your pages into different sections. Coincidentally, they are the div and section tags. Now, when shall you use what? A section tag is used to define a certain part of a topic, like a chapter or sub-topic. Whereas a div tag is used to group block-elements to later style them. I can show you how it looks in the code, when used in large platforms, just like this. In the image below, you can see that the div tag is used in 2 different places. Both are used with a class name. This class name helps us when we are trying to call it in the styling part. In the first one, it is helping us group together an image and a hyperlink. These 2 elements are for the logout button on the Progress Page.



Inserting Images: To add images into your HTML project, it is very straightforward and easy. A small simple tag called img will be used to say that we want an image here. But where will the image come from? For that, we use a src (source) tag to help the HTML fetch the image. The src must be written before the closing of the tag. In the image below, you may wonder why there isn’t a closing tag for the image. This is because this tag, and a couple others don’t require an additional tag to close them. These are called semi-closure tags. The code is seen below with the output.





Inserting Hyperlinks: If we want to redirect some content to another page or website, we must hyperlink that content. This too is quite simple for most content. We use a tag called a, which stands for anchor. This tag is also a semi-closure tag. When we say a, the HTML knows that it must be linked, it links the website once the website link is provided with the href attribute. This tag must be around the information you want to be hyperlinked. In the image below, you see the text is hyperlinked to google.com.


Creating a table: Creating a table in HTML is a little more complex than other tasks, but don’t worry because we have your back! First, we define the table with a table tag. Inside this table tag, we define the rows, with tr (table rows). Inside the tr tags, we define the th tags, which stand for table headings. We can add or remove as many rows and heading as required. I have created a table for my deadlines. See it below.





Creating Lists: In HTML, creating lists, is also very easy. You may think that we are calling everything easy, but, once you get the hang of writing the tags and its structures, everything because flawless. For writing lists, there are 2 different ways. One is with the ol tag and the other with the ul tag. The first one is for ordered lists, like a numerical list. The other is unordered list, for bluet points. Both of them define the type of list, but the content inside the list has to be written within a li tag, which stands for list item.
