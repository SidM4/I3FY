Introduction to CSS

CSS Format: CSS, stands for Cascading Style Sheet. This is a type of property that is associated with the HTML tags. CSS helps provide the aesthetic to the HTML. To keep it short, CSS is all about the look and feel of the website. Now, there are 3 different places to write the CSS. 2 of them are in the same document as the HTML and one is external. Let’s talk about the 2 first. There is inline CSS and internal CSS. Inline CSS is when we write the styling properties within the same line as the HTML tag. We write inline CSS after opening a style attribute within the HTML tag that has to be styled. Next is internal CSS, this is when the styling properties are written in the same file as the HTML, but at the top of the page within the head tags. Inside of the head tag, open the style tag as well, when we write the CSS internally. The last method is called external CSS. For this, we create a separate file with a file extension of .css. This lets the IDE know that the information within the document is in CSS. In the external CSS, we do not have to define any style tags as we can begin writing the styling properties immediately.

Inline CSS Example: When writing inline CSS, there is a basic structure that has to be followed. You go into the opening tag of the HTML element that you want to style, you open a style attribute using an equals to sign and quotation marks, then you write down one property followed by a semi-colon, closing with a colon before a new CSS property. Internal CSS: Between the head tags, we have to insert a style tag. Inside the style tag, we begin writing the CSS. It is quite a bit different to inline CSS as this one is a bit more spread out. First of all, you use the class or id name and call it out in the CSS. Use a hash for id’s and a dot for classes. Once you give the class, you open up curly brackets, and begin writing the CSS inside of there. You can see the code for the same output above.

Once you create this, go into the head tag of the HTML page and open up a link tag.
