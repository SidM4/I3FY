Reflections

Now that he is all ready and now is the time for some reflections..

25 Views

Through the project phase, there were several areas which I could have improved and due to technical challenges and constraint of time, I have parked the same. In the feedback phase as well, there were a few ideas that came up which we haven’t incorporated. Both can be captured in further phases of the application. Right now, I have built the application as a native website, however in the future, I would like to take this to a web server and that would allow several features to be activated – like user profile-based course allocation and tracking, uploading of assignments, dedicated responses, chat help etc. These would make the application more professional and the training experience more customised. Additionally, having completed my first pass of course content and blogs update, I realized I could do a lot more detailing on the journey. This is something which I have already started updating and would be part of my final project submission.

My Reflections & What Next:
Developing I3FY has been interesting. There were parts that I loved and then there was Error Checking! I did not like to sit there for hours, reading each line of colourful, complex code to find one little semi-colon or space missing. But it is part of the whole journey, right? Over the course of these 2 months, I have realised that there is so much more to developing a Web Application than just long, colourful lines of code. Going through the design cycle by starting with creating rough sketches, to laying out mind maps for each page, to writing large bits of content.

Learnings

  • The more you detail out your plan, the clearer the outcome will be.
  • Spend time on your design as much possible so that you do not need spend time in changing your code.
  • Give attention to syntax and commands – a small miss while coding, will mean a lot of time lost debugging
  • Test exhaustively, a detailed plan, a clear design, an articulate User Interface, and a large code base is no good if links are broken
  • Take regular feedback, it will only improve the outcome

    • This journey has taught me loads. I would be lying if I said I was not grateful for this experience. I have grown a great interest in programming and soon, I would like to begin learning a range of different languages, from C++ to Python to Java, etc. This journey is far from complete. I am going to continue my courses in Computer Science and take up the subject at various stages in my education. I aspire to be a great software developer in the future!
