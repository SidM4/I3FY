Responsive Web Design

To use the same web-project across a range of devices, we need to add something called responsive code. This type of code enables the website to run slightly differently on each device. In most cases, the responsive code manipulates the height and width of the elements to match the dimensions of a standard mobile phone. Now, as there isn’t any change in the format of information, it is a styling technique. Responsive Code is written in the CSS file. Here is an example of responsive code.



In this image, we see that the first line of code shows what this code is for. It states simply that the code below is for screens with maximum width of 600px, which is the standard dimension for most mobile phones. Once the first line is defined, it is just standard CSS code to alter some properties. However, 1 thing to keep in mind is the arrangement of curly brackets. There is one that opens after the first line, all responsive CSS must be written inside of this is the standard form. If the arrangement of curly brackets is right, there should be a change in colour depending on the IDE that you use.
